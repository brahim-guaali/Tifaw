# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Email **guaalibrahim@gmail.com** with details
3. Include steps to reproduce if possible

You should receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

Tifaw runs locally and does not transmit data to external servers. Security concerns are primarily around:

- Local file access permissions
- Ollama API communication (localhost only)
- pywebview window security
